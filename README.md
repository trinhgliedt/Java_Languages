# Java_Languages
A Java full stack application for a computer language dashboard.

● name: Min char 2, Max char 20

● creator: Min char 2, Max char 20

● currentVersion: Cannot be empty

Tasks:

● The GET /languages route should show a table of all the languages with links to edit and delete the languages; also include a form for creating new languages.

● The POST /languages routes creates a language.

● The GET /languages/{id} route returns the specific language and details.

● The GET /languages/{id}/edit route shows the edit page to of a specific language

● The PUT /languages/{id} route updates a language

● The DELETE /languages/{id} route deletes the specific language at that id.
